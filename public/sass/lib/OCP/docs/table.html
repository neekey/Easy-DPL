兼容性：IE6+ （具体兼容性看下方各状态说明）

最佳实践：table各元素的使用规则： 
- table
    - caption (必须紧跟在table后面）
    - colgroup（用来控制列宽，以及列维度的通用样式）
        - col
    - thead
        - tr th
    - tfoot（table数据的summary，会被渲染在底部）
        - tr td
    - tbody
        - tr td
    - tbody .. 允许出现多个tbody，但是其他元素只有一个

默认table/具有渐变色背景header/带有一个tfoot

为thead添加类.table-style-header：渐变兼容性：IE67( IE89只有灰色背景 )
